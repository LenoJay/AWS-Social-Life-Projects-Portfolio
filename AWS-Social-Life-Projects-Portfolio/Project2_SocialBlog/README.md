# 📝 Social Blog Platform

A serverless blogging site where users share updates, stories, and media with their circles. Combines dynamic content storage with real-time interactivity.

## ✅ Features
- Authenticated blog posts and comments
- Media storage (photos, clips)
- API Gateway + Lambda for CRUD
- RDS for structured data

## 🚀 AWS Services Used
- EC2
- Amazon RDS
- Amazon S3
- AWS Lambda
- API Gateway
- Amazon EFS
- AWS Cognito
- AWS Amplify (frontend hosting)
